This is a [Giter8][g8] template for zio-spark.


You can use Gitter 8 to create an example application, with all the dependencies.

For Scala 2.13
```bash
sbt new univalence/zio-spark.g8
``` 

For Scala 3
```bash
sbt new univalence/zio-spark.g8 --useScala3=true
```
